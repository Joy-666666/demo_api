# demo_api
中文： 一个使用 Ruby on Rails 构建的极简 API 示例项目，包含基础的用户管理接口与完善的错误处理机制，并附带 RSpec 请求级测试示例，适合初学者理解 Rails API 最小实现方式。  English: A minimal Ruby on Rails API example featuring basic user management, clean error handling, and RSpec request-level tests. Designed as a simple and educational reference for learning Rails API fundamentals.
